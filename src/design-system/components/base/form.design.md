## 录入结论

表单先作为基础规则组件录入，原因是本次要沉淀的是表单项结构、基础间距、上下 / 左右布局和表单内基础控件组合规则。
业务样板间不放在基础组件页，后续在“业务样板间 > 表单”下逐个录入。

标题组件已作为 `SensSectionTitle` 单独录入，不直接并入表单。Figma 中标题已有独立页面和 component set，并包含通用、营销云 / 分析云 / SDH 专用与大小尺寸变体；表单页只描述标题在表单项中的引用关系。

## Figma 提取摘要

- 表单 component set 包含 `标题类型=上标题` 与 `标题类型=左标题`。
- 表单内已覆盖输入框、数字输入框、文本域、选择器、单选组、复选组、滑动输入条、上传等控件类型。
- 上标题结构中，标题到控件的垂直距离读到 8px。
- 左标题结构中，标题到控件的水平距离读到 16px。
- 纵向表单间距读到 40px / 28px / 20px / 16px / 8px / 4px，均可映射到现有 SensD spacing token。
- 40px 用于二级标题分组之间、内容区到页脚按钮区。
- 28px 用于三级标题分组之间。
- 20px 用于组件和组件之间，排除基础表达式内部间距。
- 16px 用于标题栏 tab、sub tab、表单可使用标题、分割线、Alert、步骤条与内容区之间。
- 8px 用于联动关系的组件之间、四级标题和组件之间、组件多排等关系。
- 4px 用于辅助文案到关联内容、报错到关联内容。
- 横向表单间距读到 24px / 16px / 8px / 4px，均可映射到现有 SensD horizontal spacing token。
- 24px 用于组件与组件之间的横向间距。
- 16px 用于四级标题到组件、按钮之间、卡片之间、分割线之间。
- 8px 用于辅助文案 / 正文到内容、联动组件之间。
- 4px 用于图标到内容之间。
- 单个录入组件尺寸附录明确：组件宽度根据界面自由组合，不强制齐宽；不得小于组件最小值。
- 表单标题顺序明确为标题文案、帮助图标、选填文案。
- 必填项不展示红色星号，`required` 只保留语义和校验用途；视觉上只展示选填项。
- 表单项标题保持字段 label 语义，但视觉使用 Typography 四级标题样式：`font-size/m` + `line-height/m` + `font-weight/medium`。
- 选填文案与标题同字号同高，使用 `text-sub-color-transparent` @58% + `font-weight/regular`。
- 左右布局标题过长规则：标题文字最大宽度 112px（8 个中文字），超出省略；帮助图标、选填文案和附加信息不折行、不省略；Tooltip 基础组件未正式录入前，预览先使用原生 `title` 显示完整标题。
- 常规左右布局标题使用 32px 对齐高度；带辅助文案 / 联动内容 / 嵌套内容的单选复选，以及文本域，不额外补 32px，改为顶部对齐。
- 无辅助文案、无联动内容的单行单选组 / 复选框组，整组选项使用 `size/component-height/m` 的 32px 对齐盒；选项内图标和文字垂直居中。
- 选项自带辅助文案的单选组 / 复选框组来自 Figma `2378:29951` / `2378:29995`：标题和整组组件顶部对齐，不补 32px；选项结构为主行 22px、辅助文案 18px，中间使用 4px 间距。
- 帮助图标节点为 16px，已转为 `currentColor` SVG 并录入 `SensIcon name="help"`。
- 表单项辅助信息和字段报错共用同一信息槽；报错出现时替代辅助信息，不叠加展示。
- 报错信息结构为图标 + 文案；图标使用 `SensIcon name="feedback-error"`，文案和图标颜色使用 `warning-color`。
- 报错出现后，表单项高度自适应向下撑开，后续表单项自然下移。

## 设计边界

- 表单内的单选组、复选框组直接复用基础组件，不在 Form 中重画状态。
- 表单内报错信息属于表单项 meta；控件自身的报错视觉仍回到 Input / Select / TextArea 等组件。
- 帮助图标来自 SensD icon registry；表单标题使用 `SensIcon name="help"`，不使用导航帮助中心图标替代。
- 报错图标来自 SensD icon registry；表单项报错使用 `SensIcon name="feedback-error"`，不直接使用 antd 状态图标。
- 表单样板间中涉及锚点、步骤、模态、卡片、表格、滚动、联动等内容，后续按样板间处理。
