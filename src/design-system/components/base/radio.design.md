## 录入结论

单选框作为基础组件单独录入。它与复选框共享设计文档，但不合并代码和页面，避免多选、半选、互斥选择等规则混在一起。
单选组也归入基础组件，用于表单中的互斥选项组合；表单标题、校验和选择后的关联结果后续由表单规则接管。

## Figma 提取摘要

- 控件尺寸为 16px。
- 单选框圆角使用全圆。
- 控件与文案间距为 8px。
- 主文案为 14 / 22，辅助文案为 12 / 18。
- 取值态包含未选中、已选中。
- 交互态包含默认、悬停、点击、禁用、禁用悬停；只读态在当前基础组件 P0 暂不作为业务态展开。
- 选中内圆点以组件内部形态实现，不录入 Figma bitmap。
- 单选组横向选项间距为 `spacing/horizontal/6x`，单项内部仍为 `spacing/horizontal/2x`。
- 单选组节点注明：带辅助信息或只读状态时，需手动去掉上下间距。

## 设计边界

- 使用功能色的选中、悬停、点击态跟随 Functional Skin。
- 默认边框、文字、禁用等中性色不随功能色改变。
- 帮助图标不是单选框本体，必须来自 SensD icon registry；未确认前保持 Missing。
- 图示中的“关联结果 / 选择后展开 / 置灰关联区域”不属于本次基础组件，后续进入表单基础规则或表单样板间。
