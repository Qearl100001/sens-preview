## 一、概念　✅

应用于用户需要在同一类型的选项内容中，选择一项或多项内容。

### 1. 基础构成

- **框体 Select Box（非必须）**：承载所选结果的容器（结果分为文字 / 标签）；激活并展现承载选项内容的菜单组件。✅
- **占位符 / 默认值**：帮助用户理解当前需要操作的含义；默认为「请选择」；结果避免使用「不限」「全部」等非明确结果的文案；字数建议不超过框体宽度。✅
- **下拉图标 Icon**：表现菜单展现与否的标志（展开、收起、加载），默认是「下拉三角」；可被临时替换成「搜索」「重置」「加载」图标。预输入型选择器不带下拉三角（多标签场景节省横向空间）。
  - 箭头（展开/收起）✅ · 清空 × ✅ · 加载 🔶（透传 antd 默认、非显式「三角↔加载」切换）· 激活态换搜索图标 ⏳ · 预输入型去三角 ⏳

### 2. 使用规则

- 常用于「表单填写」和「数据筛选」。
- 有很强的业务语义才可在框体左侧使用表意 Icon。
- 展开的下拉菜单默认贴左显示。
- 根据自身业务场景，选择单选、多选、级联场景，遵守相对应的规则。
- 选择路径（官方判断路径图）：
  - 下拉菜单 → 单选下拉（平铺：基础 / 简约-无边框 / 复合-前后缀 / 组合-带筛选标签·数据分桶；树）
  - 下拉菜单 → 多选下拉（平铺：个数型-已选择 X 项·仅存在于表格筛选行 / 展示型-多标签；树 ⚠️ 用穿梭框代替）
  - 级联菜单 → 单选 / 多选 / 带搜索
  - 预输入型选择器 → 常见于表达式

### 3. 边界限制

- **框体宽度**：min 建议值 128px（6 个中文字符）；max 建议值 600px（接近 40 个中文字符）。区间内可按业务 / 表单表格规范决定自适应或固定宽度，内容超出省略显示。⏳（控件 / 高度已还原；min/max 宽度约束未在组件落盘，仅预览 demo 的 inline style 出现过）
- 结果展示「标签」的选择器建议高度：100px（3 行半），超出框体内滚动展示；业务需要可高可矮。⏳
- 结果展示「标签」的内容超过几百条时，建议使用穿梭框等代替。⏳
- 所有超出省略展示的内容，鼠标 Hover 出现 Tips 展示全部内容（动效：立即显示）。⏳
- Hover 选择器延迟 2s 后出现 Tips（若选择器前无标题则立即显示）；Tips 文案：选择 + 名词。⏳

## 二、选择器的搭配场景

> 各场景标注「下拉菜单：无操作区 / 有操作区」对应被召唤的浮层形态；浮层面板规范见 `dropdown-menu.design.md`，本文只写触发框侧与搭配关系。

### 1. 下拉菜单

#### 1.1 单选（结果：文字）

> 类型：实时选择｜结果呈现：文字｜下拉菜单：**无操作区**

**1.1.1 平铺结构**

- **1）基础型**　✅　无 Icon。选项 ≤ 5 项建议用 radio 平铺（列表页筛选行除外）；选项为简单熟悉数字时不应用选择器承载。
- **2）简约型**　⏳　无边框，用于空间不足的狭小空间（配置面板、单元格内）。不能用在列表页筛选行、不建议用在表单内；不支持图标；仅单选。
- **3）复合型**　🔶　选择器 + 输入框 / 搜索（前缀 / 后缀 / 前+后缀）。只支持单选；都不支持图标。常规 / 简约两套兼容矩阵见官方稿。（组成件选择器 / 输入框 / 搜索均已有，可业务组合实现；无独立"复合选择器"封装，工程 ⏳）
- **4）组合型**　⏳　标签式选择器 + 操作标签（数据分桶 / 维度选择）+ 删除图标。只支持单选（平铺、树）；必须是「添加方式」出现才可带删除图标；筛选标签热区上/下通顶/底。

**1.1.2 树结构**　⏳

- 不可单独选择父级节点；只能搭配基础选择器；级联与树尽量用树。
- 结果通常展示父子级关系（Hover 显层级）；可选父级 / 不可选父级 / 单选树搜索（搜索时选项下展示所在路径）等子场景。

#### 1.2 多选（结果：文字、标签）

> 类型：非实时选择｜结果呈现：文字、标签｜下拉菜单：**有操作区**

**1.2.1 平铺结构**

- **1）个数型（结果：文字）**　⏳　「已选择 N 项」，Hover 显示已选列表；最大 999+ 项。仅用于表格筛选行 / 表单内狭小筛选区域。支持重置（仅表格筛选行可用，默认不开启）。
- **2）展示型（结果：标签）**　⏳　多标签展示；删标签 ↔ 下拉对应项取消勾选，下拉选项点「完成」才回显；多行滚动（特殊才一行省略）；不支持重置和图标；选项过多用穿梭框。用于无层级多选（表格筛选行 & 狭小筛选区不可用）。

**1.2.2 树结构**　⏳

- 父级不可选；删标签 ↔ 取消勾选、点完成回显；仅支持「不可单独选父级」（可选父级建议用穿梭框）；父级全选时选择器内不显父级标签，Hover 显父子级（"/"连接）。

### 2. 级联菜单

- **2.1 单选（结果：文字）**　⏳　实时选择｜无操作区。多列展开；用于层级较少（≤ 3 级、父级固定）的场景。
- **2.2 多选（结果：标签）**　⏳　非实时选择｜有操作区。（官方标注"不定规范"）
- **2.3 特殊 - 搜索场景**　⏳　搜索默认自带并开启；框体直接输入触发搜索；搜索时多选下拉隐藏全选功能。

### 3. 预输入

- **3.1 多选（结果：标签）**　⏳　实时选择｜无操作区。用于输入下拉中不存在的值（最常见表达式-添加筛选条件）；框体内支持搜索；无下拉三角；实时生效。
- **3.2 单选（结果：文字）**　⏳　实时选择｜无操作区。单次输入不存在的值、输入形成新选项；与输入框区别在于下拉呈现后台可选数据、输入形成新选项；框体内搜索；无下拉三角。

## 三、重置机制

仅「列表页 - 筛选条件」时使用：Hover 到框体时，下拉三角变为重置图标。

- **1. 单选**　⏳　选后 Hover 显清空 / 重置；热区 = 单选区域 + 重置区域（间距 4px）。单选若没有可为空 / 回到默认值的场景，不推荐提供重置。（普通清空 clearable × 已还原；"仅列表页筛选时 hover 三角变重置图标 + 4px 重置热区"完整语义未还原）
- **2. 多选**　⏳　表格筛选指定的唯一样式（表单多选用「多选-展示型」）；选后 Hover 显重置；热区 = 多选区域 + 重置区域（间距 4px）。
