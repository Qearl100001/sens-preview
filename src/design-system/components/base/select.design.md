# 选择器 Select · 设计规范（给设计师）

> 这份讲「用什么、为什么、规格多少」，面向设计与评审。
> 「用 antd 怎么实现」见研发文档 `select.md`，本文不涉及代码/组件名。
> 源自 Sens.Design 选择器规范 v2.1 · **R3**：选择器触发框。

> 浮层面板见 `dropdown-menu.design.md`。

## 触发框（R3）

用户点击的**收起态字段**：展示已选文案或占位符，右侧下拉箭头；可搜索时点击后弹出上方 R1/R2 浮层。

### 尺寸

| 属性 | 规格 |
|---|---|
| 控件高 | **32px**（**无小尺寸**，不暴露 `size="small"` / `controlHeightSM`） |
| 圆角 | 与输入框一致（**4px**） |

### 边框与背景

与单行输入框同源：默认浅描边、悬停主色描边、聚焦深主色描边 + 外环、禁用灰底浅线。

### 文案

| 元素 | 规格 |
|---|---|
| 已选 | 正文色 |
| 占位符 | 禁用灰 @30% |
| 占位文案 | 「请选择」或业务定制（走 i18n） |

### 箭头

| 属性 | 规格 |
|---|---|
| 图标 | `ChevronDown` / `ChevronUp` **双图标互斥**（与搜索分类统一），**非**单 SVG `rotate(180deg)` |
| 尺寸 | **16×16**，与文字垂直居中 |
| 颜色 | 中性图标色；**悬停/点击不变绿** |
| 展开 | 收起 `down` / 展开 `up` |
| 禁用 | 禁用图标色 |

### 清空

| 属性 | 规格 |
|---|---|
| 默认 | 不显示（`clearable` 默认 `false`） |
| 出现 | `clearable` 且有选中值，**悬停框**时出现 × |
| 点击 | 清空选中、关闭浮层、重置浮层内搜索 |
| 位置 | 盖住右侧箭头 |

### 警告

框内红色菱形：与输入框一致，置于 `suffixIcon` 组内——**箭头在左、菱形在右**，间距 **4px**（`spacing/1x`）。

### 与浮层

点击字段 → 弹出选项浮层（R1/R2）→ 选中一项 → 回填字段文案并关闭浮层。

## 变体矩阵（R3 预览）

```text
维度：无警告 / 框内警告 × 未选 / 已选   ← 触发框仅 32px，无大/小两档
列：默认 / 悬停 / 聚焦 / 禁用 / 禁用悬停
```
