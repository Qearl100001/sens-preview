# 下拉浮层 Dropdown Menu · 设计规范（给设计师）

> 这份讲「用什么、为什么、规格多少」，面向设计与评审。
> 「用 antd 怎么实现」见研发文档 `dropdown-menu.md`，本文不涉及代码/组件名。
> 源自 Sens.Design 选择器规范 v2.1。
> **R1**：浮层容器 + 选项行。**R2**：浮层内搜索 + 内容六面。

## 范围

### R1（已完成）

浮层容器 + 选项行 10 格。不含选择器触发框 → 见 `select.design.md`。

### R2（本文展开）

浮层内搜索与内容区六面。仍不含：

- 选择器触发框 → 见 `select.design.md`
- 多选 / 分组 / 级联
- **统一异常状态组件**（页面级、完整矩阵 — 整体滞后）
- **网络异常**（`4381:14801` — 归未来异常状态规范）

## 浮层是什么

用户点击选择器字段后弹出的**选项列表面板**。单列文本选项；已选行右侧有**勾选标记**。

**术语**：设计稿里的「激活 / 选中」= **已选中行**的默认静止态（浅功能色底 + 勾选），**不是**与「默认 / 悬停 / 点击」并列的第六列。

## 浮层容器

| 属性 | 规格 |
|---|---|
| 背景 | 白 |
| 圆角 | 4px（`radius/m`） |
| 投影 | D4↓（浮层级投影，与悬浮按钮同档） |
| 上内边距 | **6px** |
| 下内边距 | **10px** |
| 左右内边距 | **0**（通栏：行底色铺满浮层宽度） |

Figma `17767:72632`。

## 选项行

| 属性 | 规格 |
|---|---|
| 行高 | **34px**（浮层专有，与触发框 32px 控件高无关） |
| 尺寸 | **仅单一尺寸**，浮层无大/小两档 |
| 布局 | **通栏**；文案/勾选在行内水平缩进 **12px** |
| 行内上下内边距 | **6px**（配合 14px/22px 正文凑满 34px 行高） |

## 选项底色

### 未选中行

| 状态 | 背景 |
|---|---|
| 默认 | 白 |
| 悬停 | 中性灰 hover 底 **6%** |
| 点击（鼠标按下瞬间） | 中性底 **8%** |
| 禁用 · 默认 | 白 |
| 禁用 · 悬停 | 中性灰 hover 底 **6%** |

### 已选中行（浅功能色底 · 随主题色换肤 · 右侧勾选）

| 状态 | 背景 |
|---|---|
| 默认（= 设计稿「选中/激活」静止态） | 浅功能色底 · 默认档 |
| 悬停 | 浅功能色底 · 悬停档 |
| 点击 | 浅功能色底 · 点击档 |
| 禁用 · 默认 | 白 |
| 禁用 · 悬停 | 中性灰 hover 底 **6%** |

### 文字与勾选

| 元素 | 颜色 |
|---|---|
| 选项文字（未禁用） | 正文色 |
| 选项文字（禁用） | 禁用灰 |
| 勾选标记（已选中 · 默认/悬停/点击） | 中性图标 · 01 主要 |
| 勾选标记（已选中 · 禁用/禁用悬停） | 中性图标 · 04 禁用 |

## 变体矩阵（R1 预览）

```
行 · 选中：未选中 / 已选中
列 · 状态：默认 / 悬停 / 点击 / 禁用 / 禁用悬停
```

共 **2 × 5 = 10** 格。按分组并排，不拍平、不合并。

## 浮层内搜索（R2）

### 什么时候出现

选项较多、需在浮层内快速定位时，在选项列表**上方**嵌入搜索框。

### 搜索框规格

沿用 Sens.Design **搜索规范 · 实时·简约**：

| 属性 | 规格 |
|---|---|
| 交互 | **实时搜索**；输入停顿约 **300ms** 后出结果 |
| 宽度 | 铺满浮层（**通栏**；分割线到边） |
| 行高 | **32px**（简约搜索交互区） |
| 文案内缩 | **12px**（与选项行文案左缩进对齐；在输入框 affix 内） |
| 至统计间距 | 底部分割线至「共 N 条」**6px** |
| 统计至首条 | **0px** |
| 浮层顶（可搜索） | **0px**（搜索贴顶） |
| 底部分隔 | 搜索框**自有**底部分割线；**不再**额外加横线 |
| 清空 | 有有效输入时展示清空控件 |
| 占位符 | 「请输入搜索内容」或业务定制 |

### 布局

```text
┌─────────────────────────┐
│  [🔍  请输入搜索内容    ] │  ← 通栏；文案/图标内缩 12px；行高 32px
│─────────────────────────│
│  ↕ 6px                  │
│  共 N 条数据              │  ← 03_辅助色；仅未搜索 / 有结果
│  选项 A              ✓   │  ← 统计与首条间距 0；对勾距右 12px
│  …                       │
└─────────────────────────┘
```

## 内容区六面（R2）

| 面 | 何时 | 用户看到 |
|---|---|---|
| **未搜索** | 打开浮层，未输入，有数据 | 全量列表 + 「共 N 条数据」 |
| **搜索中** | 已输入，结果未出 | 列表区 **小 spinner** |
| **有结果** | 搜索完成，有匹配 | 列表 + 主题色高亮 + 「共找到 N 条」 |
| **无结果** | 搜索完成，零匹配 | **暂无结果**小插图 + 文案 |
| **加载中** | 选项数据尚在加载 | **小 spinner**（可选「加载中」） |
| **暂无数据** | 源数据为空、未搜索 | **暂无数据**小插图 + 文案 |

源加载失败（非搜索失败）另用 **加载失败** 插图面（不进六面矩阵）。

### 统计文案

| 场景 | 文案 |
|---|---|
| 未搜索 | 共 **N** 条数据 |
| 有结果 | 共找到 **N** 条 |

统计文案色：**中性色/文字/03_辅助** `@text-sub-color-transparent` @ **58%**。

### 关键词高亮

- 匹配段用**主题色**字，随换肤；不加底色、不加粗
- 全部匹配段均高亮
- 首版：完整文案；省略段内高亮后续

### 空态插图（非页面级 · 特殊尺寸）

三种插图，源自 **非页面级异常状态** `4372:25735` · **特殊尺寸**：

| 用于 | Figma | 主文案 | 副文案 |
|---|---|---|---|
| 搜索无匹配 | `4381:14824` | 暂无结果 | 未找到结果，请重新输入 |
| 源数据为空 | `4381:14868` | 暂无数据 | 暂无数据，请 **添加** |
| 源加载失败 | `4381:14891` | 加载失败 | 数据加载失败，请 **刷新** |

**不在 R2**：`4381:14801` · 网络异常。

空态块：插图 **50×50** · 文案 **140px** 宽 · 间距 **12/4** · 内边距 **20**。

- **主标题**：14px · 中性色/文字/01_主要
- **辅助文案**：**12px** · **中性色/文字/03_辅助** `@text-sub-color-transparent` @ **58%**（无结果描述、加载失败/暂无数据的前置句）
- **操作链接**（刷新 / 添加）：`colorLink`，12px

### 「加载中」与「搜索中」

| | 加载中 | 搜索中 |
|---|---|---|
| 含义 | 列表还没准备好 | 正在按关键词检索 |
| 视觉 | 小 spinner | 小 spinner |

二者均**不用**空态插图。

## 交互细则（R2）

| 行为 | 规则 |
|---|---|
| 输入 | 实时过滤；300ms 防抖 |
| 清空 | 回未搜索全量（或暂无数据） |
| 关闭浮层 | **重置搜索** |
| 选中 | 关浮层并重置搜索 |
| 搜索失败 | 归 **无结果** |
| 源加载失败 | **加载失败** 面 |

## 变体矩阵（R2 预览）

```text
列：未搜索 / 搜索中 / 有结果 / 无结果 / 加载中 / 暂无数据
```

与 R1 选项行 10 格分开展示。
