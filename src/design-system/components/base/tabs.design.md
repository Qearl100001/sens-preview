# 标签页 Tabs · 设计规范（给设计师）

> 这份讲"用什么、为什么、规格多少"，面向设计与评审。
> "用 antd 怎么实现"见研发文档 `tabs.md`，本文不涉及代码/组件名。

## 什么时候用标签页
在同一**页面/模块**内，把**同层级**的多个标题聚拢展示。原则：
- **同层级**：一组标签页彼此是平级的。
- **彼此互斥**：每个标签页管辖的内容（信息/操作/状态）应当互斥。
- **禁忌**：强烈不建议在「编辑/创建」场景用标签页——它有"并立"的语义，提交时会让对象/信息显得不准确。

## 三种类型（按场景选，这是选型核心）
| 类型 | 用在哪 |
|---|---|
| 基础标签页 | 页面级同层级内容折叠，最常见 |
| 页签标签页 | 卡片级、像浏览器/编辑窗口那样可增删的标签 |
| 胶囊标签页 | 编辑器里切换面板（如 通用/高级/策略）。**胶囊标签页不是按钮** |

## 通用交互原则
- 切换标签页：默认**保留**浏览状态（分页/筛选/位置）；默认**放弃**编辑状态。
- 不支持点击"当前所在"的标签页来刷新内容。

## 各类型规格
**基础标签页**
- 一组至少 2 个标题（特殊权限场景可只 1 个）。
- 标题不可折行，建议 ≤8 个中文字（128px），超出「…」省略 + 悬停 tooltip。
- 选中项：绿色文字 + 底部高亮条滑到当前位置。
- 标题可搭配徽标强调变化。
- 标签过多：最右加「全部」，点开展示全部并定位到当前。

**页签标签页**
- 构成：标题 + 删除/关闭 + 添加 + 槽位。
- 可双击编辑当前页签标题、可拖拽排序、可设数量上限；只剩 1 个时隐藏删除。
- 标题 ≤8 中文字，超出省略。过多时折叠为下拉并定位当前。

**胶囊标签页**
- ≤5 项；每项 ≤112px，超长截断 + 悬停 tooltip。
- 选中：白底 + 轻投影 + 绿字；选中悬停字色为 hover 绿。
- **只能独立使用**，不允许和其它标签页搭配来表达层级。
